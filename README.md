# init-fifa-archive

## 项目简介

`init-fifa-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何单个域名或具体网站，旨在以静态 HTML 的形式，长期保存并展示一组自包含的网页内容。  

所有页面均为独立文件，不依赖外部数据库或后端服务，便于浏览、分享与二次利用。

## 目录说明

```
.
├── index.html          # 仓库首页/入口页面
├── pages/              # 归档页面存放目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立的归档 HTML 页面。  
- `assets/` 目录存放页面共享的静态资源，便于统一维护。  
- 每个页面应保持完整、自包含，尽量不依赖外部资源。

## 页面归档说明

- 所有归档页面均为静态 HTML，可直接在浏览器中打开使用。  
- 页面内容来源于公开可获取的信息，仅作存档用途。  
- 归档范围包括但不限于：特定主题的展示页、历史快照、信息整理页面等。  
- 每个页面文件名应具备一定辨识度，便于索引与引用。

## 维护说明

- 本仓库由维护者不定期更新，添加或修正归档页面。  
- 欢迎提交 Issue 或 Pull Request 来补充、修正或建议新页面。  
- 提交新页面时，请确保：
  - 页面为独立 HTML，无外部依赖或已内联所有资源；
  - 文件名适当且不与其他页面冲突；
  - 页面内容不违反相关法律法规与平台政策。
